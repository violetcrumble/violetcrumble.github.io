Drag and Drop Home Search Interface Prototype

I created this long ago, back in 2015. I didn't use any kind of framework - it's vanilla JS with a drag and drop and touch detection plugin. This repo can be cloned and if you go to index.html, you can search for homes by dragging the rooms on the right to the grid on the left. This was a proof of concept. If I were to redo this project today, I would definitely not use jQuery. I cringe at the way everything is in the one draganddrop.js file and not in components. However, I do like that I wrote it as a simple plugin that could be dropped into any website or landing page.
